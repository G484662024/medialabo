数当てゲーム

1〜10の値を1つを決めて、答えとする。答えを予想して、数を3回まで入力可能。予想した数と答えが一致しない場合は、入力よりも答えが大きいか小さいか表示する。3回以内に予想当てれば勝ち、それ以外は負け。

演習の成果： kazuate.html

各授業の学習事項

第1回 WWWとHTML

今回は初めてWebコンテンツを自分で作成しました。色々な知識を頭に入って、爆発しそうです。たとえば、改行とか見出しを作るとか、ULRをアップする、図を上げたりなど学べました。

演習の成果： 演習1-4：HTMLのさまざまな要素

第2回CSSによるWebページの装飾

今回はHTMLと違うCSSファイルを習いました。CSSはWebページを装飾の役割を果たしています。装飾方法は山ほどあって、どうすればいいか少し迷ってしまいました。特にプロパティ名のところはとまりました。

演習の成果： 演習2-3: ウェブブラウザ説明ページの装飾

第3回JavaScriptの基本

今回はJavaScript プログラムについて学べました。javaと凄く名前は近いですが、違うものです。最も大きな違いは、インタプリタ言語とコンパイル言語です。でも、同じ部分もあって、例えば、 console.log(...) は Java の System.out.println() のように文字列を出力します。

演習の成果： 演習3-6

第4回DOMとJavaScript

今回の授業では、HTMLでフォーム要素を設置し、JavaScriptによるイベント処理とDOM操作を学ぶことで、ボタンを押すとページが変化する動的なWebページの作成方法を習得します。

演習の成果:演習4-10:足し算
