「Webコンテンツ作成」の成果

本テーマについて

学習目的：

    HTMLはWebページの骨組みを作り、見出しや段落、画像などの要素を配置します。CSSはそのデザインを整え、色やレイアウト、文字サイズなどを調整します。JavaScriptは動きを加える、ボタンを押すと表示が変わるなどのインタラクティブな機能を実現します。

プロジェクト課題について

今回の課題では、世界各地の天気を表示するウェブページを制作しました。HTMLで検索ボックス、表示ボタン、そして11か所の地域選択肢を設置し、ユーザーが都市を選んで天気情報を取得できるようにしました。CSSを使ってページのデザインを整え、特に見やすさと雰囲気づくりにこだわりました。背景には明るい青色を選び、ページ中央には濃い青色の天気アイコンを配置しました。空を連想させる明るい青色は、利用者に爽やかで前向きな印象を与えると考えたからです。配色は何度も試行錯誤し、最終的に「天気といえば空」という発想からこのデザインに決定しました。JavaScriptを用いて、ボタンをクリックすると選択された都市の天気が表示される動作を実装しました。この課題を通じて、構造・デザイン・機能の連携が重要であることを実感しました。

プロジェクト課題で作成したページ: project.html

数当てゲーム

1〜10の値を1つを決めて、答えとします。答えを予想して、数を3回まで入力可能。予想した数と答えが一致しない場合は、入力よりも答えが大きいか小さいか表示する。3回以内に予想当てれば勝ち、それ以外は負け。 今回の課題では、JavaScriptを活用して数字当てゲームを制作しました。HTMLで入力ボックスと判定ボタンを配置し、ユーザーが数字を入力して答えを確認できるようにしました。CSSでレイアウトと配色を調整し、見やすく遊びやすいデザインにしました。JavaScriptでは、ランダムに設定した正解の数字とユーザーが入力した値を比較し、大きいか小さいかのヒントを返す機能を実装しました。これにより、ユーザーが試行錯誤しながら正解を導き出せるインタラクティブなゲームとなりました。

演習の成果： kazuate.html

各授業の学習事項

第1回 WWWとHTML

今回は初めてWebコンテンツを自分で作成しました。色々な知識を頭に入って、爆発しそうです。たとえば、改行とか見出しを作るとか、ULRをアップする、図を上げたりなど学べました。世界の天気のデータの Web アプリケーションを作ることにしました。

演習の成果： 演習1-4：HTMLのさまざまな要素

第2回CSSによるWebページの装飾

今回はHTMLと違うCSSファイルを習いました。CSSはWebページを装飾の役割を果たしています。装飾方法は山ほどあって、どうすればいいか少し迷ってしまいました。特にプロパティ名のところはとまりました。プロジェクター課題については今回習ったCSSを使って、見た目を変えました。

演習の成果： 演習2-3: ウェブブラウザ説明ページの装飾

第3回JavaScriptの基本

今回はJavaScript プログラムについて学べました。javaと凄く名前は近いですが、違うものです。最も大きな違いは、インタプリタ言語とコンパイル言語です。でも、同じ部分もあって、例えば、console.log(...) は Java の System.out.println() のように文字列を出力します。

演習の成果： 演習3-6

第4回DOMとJavaScript

今回の授業では、HTMLでフォーム要素を設置し、JavaScriptによるイベント処理とDOM操作を学ぶことで、ボタンを押すとページが変化する動的なWebページの作成方法を習得します。数当てゲームについては、予想値を4だけ出ることではなく、バラバラと改善しました。

演習の成果:演習4-10:足し算

第5回イベント処理

今回の授業では、DOMを使って要素の新規作成、要素の削除、要素の属性を追加したり変化したりなどやりました。ページ上のボタンを押すとあとで追加した内容が一気に全て表示することになりました。プロジェクト課題については、コンソールで printDom(data); を実行するとオブジェクトをページ上に表示するようになりました。

演習の成果:演習5-7

本学期のウェブ制作の授業では、HTML・CSS・JavaScriptというウェブ制作に欠かせない3つの基礎技術を体系的に学び、複数の実践的な課題を通して知識を応用する力を身につけることができました。この授業を通じて、ウェブページがどのように構成され、動作するのかという仕組みを理解できただけでなく、自分でゼロから機能的なウェブページを作り上げる達成感を得ることができました。さらに、これらのスキルが将来の学習や仕事において大きな役割を果たすということも実感しました。 まず、授業の前半ではHTMLの基礎を学びました。HTMLはウェブページの骨組みを作るマークアップ言語であり、見出し・段落・リンク・画像・表・フォームなどを記述することで、ページの基本構造を組み立てられます。特にフォーム要素の学習では、入力欄・ボタン・選択肢の設置方法を理解し、ユーザーが操作できるインターフェースを作成できるようになりました。 次に、CSSを用いることで、文字の装飾や背景色の設定、レイアウトの調整ができ、視覚的に見やすいページを作成できます。Flexboxを活用した配置や配色の工夫を行い、実用性とデザイン性を両立させる重要性を体感しました。CSSは単なる装飾のための技術ではなく、ユーザーの体験を向上させるための大切な役割を担っていることを学びました。 さらに、JavaScriptを用いてページに動的な機能を追加しました。JavaScriptによって、ボタンを押すと表示内容が変わる仕組みや、入力データに応じた処理を実行する機能を実装しました。特にプロジェクトでは、選択した都市の天気情報を表示する機能や、入力値を判定してフィードバックを返す数字当てゲームを作成し、動きのあるインタラクティブなページを完成させました。この過程で、条件分岐やループ処理といったプログラミングの基本も身につけることができました。 実践課題としては、「世界各地の天気を調べるウェブページ」と「数字当てゲーム」を制作しました。天気のページでは、検索欄・ボタン・11か所の地域選択肢を設け、ユーザーが都市を選んで天気情報を取得できる仕組みを作りました。特にデザイン面では時間をかけました。例えば、天気のページでは見やすさと雰囲気を重視し、ページ中央に濃い青色の天気アイコンを配置し、背景には明るめの青を選びました。空のような明るい青色は見る人に爽やかな印象を与え、気分を明るくする効果があると考えたからです。配色を決める際には何度も試行錯誤を行い、最終的に「天気といえば空」という発想から空の色を意識したデザインに決定しました。こうしたデザインの工夫は、ユーザーにとって直感的で心地よい体験につながると感じています。 授業では、先生から「将来社会に出て会社に入ったとき、ウェブ制作のスキルは多くの場面で必要とされる」とのお話もありました。この言葉を通じて、今学んでいることが実務に直結する大切なスキルであると理解しました。ウェブ制作の技術は、企業サイトや社内システムの開発、情報発信など幅広い分野で活用されるため、今後の進路やキャリアにおいて必ず役立つと感じています。 今回の学びを通じて、単なる知識の習得だけでなく、自分で調べ、考え、問題を解決する力がついたと実感しています。制作中にはコードが思い通りに動かない、デザインが崩れるといった課題もありましたが、調べながら試行錯誤を繰り返すことで解決できました。この経験は、今後の学習や仕事でも大きな糧になると思います。 総じて、この授業はウェブ制作の基礎から実践までを体系的に学ぶ貴重な機会となりました。HTMLで骨組みを作り、CSSで見た目を整え、JavaScriptで動きを加えるという開発の流れを身につけたことは、今後の学習やキャリア形成において確かな強みになると感じています。これらのスキルをさらに磨き、将来社会で活かせるよう努力を続けていきたいです。
